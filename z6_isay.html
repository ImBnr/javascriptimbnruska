Zadanie 6 — Temat: Zadanie teoretyczne

1) Rodzaje okienek
Okienko alert – wyświetla okienko dialogowe z informacją i przyciskiem OK.
Okienko prompt – generuje okno, w którym użytkownik może wpisać wartość; zwraca zawsze ciąg znaków (string) do programu.
Okienko confirm – służy do potwierdzenia przez użytkownika; zwraca true (jeśli OK) lub false (jeśli Anuluj).
2) Konwersji liczb
Dane wczytane przez prompt() są domyślnie ciągiem znaków. Aby wykonać obliczenia, trzeba je skonwertować do typów liczbowych. Najważniejsze funkcje konwersji:
parseFloat(zmienna) – konwersja do liczby zmiennoprzecinkowej (float).
parseInt(zmienna) – konwersja do liczby całkowitej (int). Uwaga: obcina część po przecinku i można podać drugi argument (radix), np. parseInt("10", 10).
toString(zmienna) – konwersja wartości do łańcucha znaków (string).
Przykłady: 
                        let s = prompt("Podaj liczbę:");
                        let x = parseFloat(s);   // teraz x to liczba zmiennoprzecinkowa
                        let n = parseInt(s);     // teraz n to liczba całkowita
                        let txt = (123).toString(); // "123"
3) Przy nadawaniu nazw zmiennym należy przestrzegać pewnych zasad:
Poniżej trzy najważniejsze zasady:
  1. Nazwa musi zaczynać się od litery, znaku $ lub _ i może zawierać tylko litery (bez polskich znaków), cyfry oraz $ i _. Przykłady poprawne: zmienna1, $wynik, _rozmiar. Przykłady błędne: 1zmienna, wynik gry, wysokość.
  2. Wielkość liter ma znaczenie. Rekord i rekord to dwie różne zmienne — to częsta przyczyna błędów.
  3. Nie używaj słów kluczowych JavaScript jako nazw zmiennych (np. for, if, let, class itp.).
4) Omów typy zmiennych w Javascript
W JavaScript typy nie są deklarowane wcześniej tak jak w C++; zmienna otrzymuje typ w momencie przypisania. Podstawowe typy:
  • Number – typ liczbowy. Obejmuje liczby całkowite i zmiennoprzecinkowe (w JS oba są typu Number).
  • String – typ tekstowy (łańcuch znaków).
  • Boolean – typ logiczny: true lub false.
  • Null – reprezentuje świadomie pustą zawartość (wartość "brak").
  • Undefined – zmienna zadeklarowana, ale nieprzypisana ma wartość undefined.
  • Object – obiekty, tablice, funkcje i inne struktur danych.
